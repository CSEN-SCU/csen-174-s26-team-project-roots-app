Frank · divergent prototype

From saved reels to a rough-cut trip plan

This explorer tests the “POWERED BY” idea in our product vision: multimodal-ish signals (starting with captions from your saved travel reels) become named places, then a calendar-aware draft itinerary with crude travel-time spacing.

Who it is for

People who collect travel inspiration on Instagram but stall when translating saves into a workable schedule—especially when clips mention venues obliquely or without location tags.

What problem it tackles

“Doom-saving” creates a pile of context-poor media. This flow pulls candidate spots out of reel text, keeps only what geocodes near your destination, and schedules visits with simple hours + travel buffers.

How to use it (gallery walk)

  1. Switch to Planner.
  2. Enter destination + trip dates, create trip.
  3. Load reels: Demo Lisbon set, paste an Instagram access token, or import JSON captions.
  4. Run Extract places (AI), then Build itinerary.

Instagram reality check

Meta does not expose your private Saved folder to third-party APIs. This prototype can pull recent media with an Instagram Graph token (developer setup), or you can import captions manually—closest to a “saved reels” folder for a class demo.

Never paste passwords here. Use a short-lived access token from the Meta developer flow, or stay in demo/import mode.
